# Fonts Directory

This folder contains fonts for the bot's canvas features.

## Supported Font Types
- Bangla fonts (Bengali script)
- English fonts (Latin script)

## How to Add Fonts

1. Place your .ttf or .otf font files in this directory
2. The bot will automatically detect and register them

## Recommended Fonts

### Bangla Fonts:
- Noto Sans Bengali
- Kalpurush
- SolaimanLipi
- Nikosh

### English Fonts:
- Roboto
- Open Sans
- Inter
- Segoe UI

## Font Registration

Fonts are automatically registered when canvas is available.
If canvas is not available, the bot will use text-based fallbacks.
